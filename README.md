## ✨ Фишки
- Конверт с анимацией открытия
- Конфетти (canvas-confetti) с резервным эффектом
- Летающие сердечки и пульсирующий стикер
- Персонализация имени через ссылку

## 💌 Персонализация имени
Добавь `?name=` к ссылке (имя в родительном падеже):

```
https://zxcursed667.github.io/proekt1/?name=Маши
```
