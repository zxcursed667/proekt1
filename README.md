# 🎀 Генератор открыток — proekt1

Интерактивные веб-открытки + **Telegram-бот**, который генерирует персональные открытки-ссылки.
Проект работает **без бэкенда**: всё содержимое открытки кодируется прямо в ссылке.

## 🔗 Живые ссылки
- 🌐 Открытка: https://zxcursed667.github.io/proekt1/
- 🎨 Генератор (пример): https://zxcursed667.github.io/proekt1/card/gen.html?to=Ангелины&msg=Хорошего дня!&theme=sunny
- 🤖 Бот: `@имя_бота` (после деплоя)

## ✨ Возможности
- 💌 **Анимированная открытка**: конверт с открытием, печатающийся текст, конфетти, фоновая музыка и летающие сердечки
- 🎨 **Генератор**: имя / текст / подпись / тема передаются через URL (`?to=&msg=&from=&theme=`)
- 🎭 **6 тем** оформления: розовая, солнечная, ночная, мятная, морская, золотая
- 🤖 **Telegram-бот** отдаёт ссылку, **превью открытки** и **QR-код**
- 📤 **Кнопки «Открыть» и «Поделиться»** прямо в чате
- 📱 **Open Graph** превью при отправке ссылки в мессенджерах

## 🧱 Стек
- **Frontend:** HTML, CSS, ванильный JavaScript
- **Анимации:** CSS-анимации, Web Animations API, [canvas-confetti](https://github.com/catdad/canvas-confetti), Web Audio API
- **Бот:** Python 3 + [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI)
- **QR / превью:** api.qrserver.com и image.thum.io (без ключа)
- **Хостинг:** GitHub Pages (статика) + Pella (бот, бесплатно)

## 🛠️ Как работает генератор
Всё состояние открытки живёт в адресе ссылки — бэкенд не нужен:
```
card/gen.html?to=Имя&msg=Текст&from=Подпись&theme=pink
```
Темы: `pink`, `sunny`, `night`, `mint`, `ocean`, `gold`. Добавь `&open=1` — открытка покажется сразу (режим превью).
Бот собирает эти параметры у пользователя и собирает из них URL.
